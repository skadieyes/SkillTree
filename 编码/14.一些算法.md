# 14. 一些算法

## 获得两个数组的交集

### code

```javascript
    const intersect = (nums1, nums2) => {
        const map = {}
        const res = []
        for (let n of nums1) {
            if (map[n]) {
                map[n]++
            } else {
                map[n] = 1
        }
        }
        for (let n of nums2) {
            if (map[n] > 0) {
                res.push(n)
                map[n]--
            }
        }
        return res
    }
```
